# comedero-gatos-TEC-2025
# Comedero Automático para Gatos — Ignacio Zamora Sánchez
## Descripción
Pequeño sistema que dispensa alimento automáticamente usando
<Arduino/ESP32> y un motor.
## Funcionamiento
- El motor entrega porciones controladas.
- Se puede activar con botón o temporizador.
